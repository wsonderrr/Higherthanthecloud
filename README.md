# Higher than the cloud
🐤 基于 C++ / FTXUI 实现的由AI辅助生成的飞翔小鸟游戏  
面向对象程序设计（C++）课程大作业

---

## 📝 项目介绍
- 终端实时渲染画面
- 物理重力系统 + 碰撞检测
- 分数系统 + 最高分记录
- 多线程物理更新 + UI 渲染分离
- 简洁美观的交互界面


## 🧩 环境依赖
- 支持 **C++17** 的编译器
- **CMake >= 3.20**
- 首次编译需要网络（CMake 自动下载 FTXUI）

## 🎮 操作指南
- 游戏启动后会自动初始化并进入等待状态，在控制台交互时请确保处于英文输入法状态：
- [ Space ] (空格键)：给小鸟一个向上的瞬时加速度（振翅翱翔）。
- [ R ] 键：在游戏结束（Game Over）时，一键重置所有状态并重新开始游戏。
- [ Q ] 键：在任何时候按下，安全退出游戏并返回终端。
