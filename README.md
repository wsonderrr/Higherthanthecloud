# Higher than the cloud
🐤 基于 C++ / FTXUI 实现的终端飞翔小鸟游戏  
面向对象程序设计（C++）课程大作业

---

## 📝 项目介绍
本项目是一款运行在**终端/控制台**中的飞翔小鸟（Flapp Bird）游戏，使用 **C++17** 与 **FTXUI 终端图形库**开发。
项目采用**面向对象设计**，将游戏逻辑与界面渲染分离，结构清晰、易于扩展，适合作为 C++ 课程大作业、练手项目或演示案例。

**核心特性：**
- 纯 C++ 面向对象设计
- 终端实时渲染画面
- 物理重力系统 + 碰撞检测
- 分数系统 + 最高分记录
- 多线程物理更新 + UI 渲染分离
- 简洁美观的交互界面

---

## 🧩 环境依赖
- 支持 **C++17** 的编译器（GCC/Clang/MSVC）
- **CMake >= 3.20**
- 首次编译需要网络（CMake 自动下载 FTXUI）
